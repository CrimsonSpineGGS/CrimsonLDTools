# Crimson Tool

## Setup 

Open any unity project, then go to

**Window** > **Package Manager**

The package manager window is now open. Click on the "+" icon in the upper left corner then click on **Install package from Git URL**. 

Copy and Paste the following URL : 

```
https://github.com/CrimsonSpineGGS/CrimsonLDTools
```

You can now find the Place Actor window in **Crimson Tool** > **Place Actor**. And the primitive quick action into your main toolbar on the viewport with the name **PA**. 

The tool is now added to your project. Enjoy !

PS : If errors show in the console, clear it then reopen the ToolBar or the Place Actor window. 

### Update 

If you already have the tool you can upgrade it by going into the Package Manager Window, find the package and click on **Update**. The engine will update the package for you !
